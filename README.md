# 🏋️ PinhoGym  
### Sistema de Gerenciamento para Academias

O **PinhoGym** é um sistema web desenvolvido para auxiliar no gerenciamento de academias, oferecendo controle de alunos, dashboard administrativo, gerenciamento financeiro e análise de dados.

O projeto foi criado inicialmente para ser utilizado e apresentado para uma academia da cidade, com foco em facilitar a organização administrativa, controle financeiro e gerenciamento de clientes.

Além de servir como projeto prático de desenvolvimento web, o sistema também foi pensado como uma solução real para academias de pequeno e médio porte.

---

## 🚀 Funcionalidades

- 👤 Cadastro de alunos  
- 📋 Gerenciamento de clientes  
- 💰 Controle de mensalidades e contas a pagar  
- 📊 Dashboard administrativo  
- ✅ Verificação de alunos com pagamentos em dia  
- ⚠️ Controle de pagamentos pendentes  
- 📈 Gráficos de crescimento mensal de alunos  
- 🌐 Interface web simples e funcional  

---

## 📊 Dashboard Administrativo

O sistema possui um dashboard administrativo que fornece informações importantes da academia em tempo real.

### Recursos do dashboard:
- Quantidade de alunos cadastrados  
- Clientes com pagamentos realizados  
- Clientes com pagamentos pendentes  
- Crescimento mensal de novos alunos  
- Gráficos administrativos e financeiros
